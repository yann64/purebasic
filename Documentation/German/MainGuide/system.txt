@Title Systemvoraussetzungen

  PureBasic läuft auf Windows Vista, Windows 7, Windows 8, Windows 10 und Windows 11 (sowohl in der 32-Bit- als auch in der 64-Bit-Version),
  Linux (Kernel 2.2 oder höher), MacOS X (10.15 oder höher) und Raspberry PI OS (Bookworm/Debian 12 oder höher). @LineBreak
  @LineBreak
  Falls irgendweIche Probleme auftreten, informieren Sie uns bitte: support@purebasic.com

@Section Windows
  Für viele Grafik-Funktionen wie @LibraryLink "engine3d" "3D" oder @LibraryLink "sprite" "Sprites"
  wird die DirectX9.0c Version benötigt. Windows Vista und neuer haben diese nicht standardmäßig
  installiert, weshalb Sie dies manuell nachholen müssen.
  @LineBreak
  @LineBreak
  Sie können dies downloaden und installieren mit Hilfe des "DirectX End-User Runtime Web Installer"
  hier: @InternetLink "http://www.microsoft.com/en-us/download/details.aspx?displaylang=en&id=35" "http://www.microsoft.com/en-us/download/details.aspx?displaylang=en&id=35".

@Section Linux

  Benötigte Pakete zur Verwendung von PureBasic:
  @Indent @Section
    - SDL 1.2 devel (für die 'Joystick' und 'AudioCD' Bibliotheken) @LineBreak
    - GTK 3 (für GUI-Programme) @LineBreak
    - libstdc++ devel @LineBreak
    - GCC korrekt installiert @LineBreak
  @LineBreak  
    - iodbc und iodbc-devel, um die @LibraryLink "database" "Database" Befehle nutzen zu können (siehe @InternetLink "http://www.iodbc.org" "www.iodbc.org") @LineBreak
  @LineBreak
    - libgtkembedmoz.so sollte installiert sein, damit das @@WebGadget funktioniert (wird mit dem Mozilla/Firefox Paket ausgeliefert). @LineBreak
  @LineBreak
    - libvlc-devel für die @LibraryLink "movie" "Movie" Befehle
  @EndIndent 

  Weitere Informationen finden Sie in der INSTALL- und README-Datei.

@Section MacOS X

  Zur Verwendung von PureBasic müssen Sie die Apple-Entwicklerwerkzeuge (Apple Developer Tools) installieren.
  Diese Tools finden Sie entweder auf einer der Installations-CDs oder auf der Apple Webseite: 
  @InternetLink "http://developer.apple.com/" "http://developer.apple.com/" 
  @LineBreak
  @LineBreak
  Bitte achten Sie darauf, die neueste Version der Tools (z.B. XCode) - welche für Ihre OS Version
  benötigt wird - zu verwenden!
  @LineBreak
  Die 'Commandline Tools' müssen nach der Installation über XCode installiert werden.
