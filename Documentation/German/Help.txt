@Library Help

@Overview
  Die Hilfe ist ein sehr bedeutender Bestandteil jeder Art von Software. Sie gibt den Endanwendern
  die Möglichkeit, die Software-Funktionen schnell und einfach zu entdecken.
@LineBreak
  Die PureBasic Help-Funktionen ermöglichen dem Programmierer, Standard Hilfe-Dateien mit globaler
  und kontextbezogener Unterstützung anzuzeigen.
@LineBreak
@LineBreak
  Auf Microsoft Windows gibt es zwei verschiedene Formate von Hilfe-Dateien, welche unterstützt
  werden: Das ".hlp" Format (was das alte Format ist) und das ".chm" Format (was das neue Format
  ist). Das ".chm" Format basiert auf HTML, was für "Hyper Text Markup Language" steht.

@CommandList

@ExampleFile All Help.pb

@SupportedOS Windows, OSX

;--------------------------------------------------------------------------------------------------------

@Function CloseHelp()

@Description
  Schließt das zuvor mit @@OpenHelp geöffnete Hilfe-Fenster.

@NoParameters
@NoReturnValue

@SeeAlso
  @@OpenHelp

@SupportedOS Windows, OSX

;--------------------------------------------------------------------------------------------------------

@Function OpenHelp(Dateiname$, Thema$)

@Description
  Öffnet ein Hilfe-Fenster und zeigt dieses an.

@Parameter "Dateiname$"
  Der Name der zu öffnenden .chm oder .hlp Datei.

@Parameter "Thema$"
  Der interne Name der darzustellenden Seite.
  Wenn der Parameter 'Thema$' kein leerer String ist, stellt die OpenHelp-Funktion
  direkt die angegebene Seite dar (sehr nützlich für Online/kontext-sensitive Hilfe).

@NoReturnValue

@Example
@Code
  OpenHelp("help.chm", "index.htm")
  OpenHelp("help.chm", "print.txt")
@EndCode

@SeeAlso
  @@CloseHelp

@SupportedOS Windows, OSX
