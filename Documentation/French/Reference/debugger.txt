<a name="CallDebugger"></a>
@FormatEndIf

@Syntax

  @Keyword CallDebugger
  
@Description    

  Appel du "débogueur" et arrêt immédiat du programme au point courant du code. 
@LineBreak
@LineBreak

@FormatIf HTML
<br><hr><br>
<a name="Debug"></a>
@FormatEndIf

@Syntax

  @Keyword Debug <expression> [, NiveauDebug]

@Description    

  Affiche la fenêtre DebugOutput et le résultat correspondant. L'expression peut être toute 
  expression valide en PureBasic, de forme numérique ou chaine. Un point important est que 
  toute commande @Keyword Debug et les expressions associées sont totalement ignorées (non compilées) 
  si le débogueur est désactivé. Cela singifie qu'il n'est pas nécessaire de passer les instruction 
  @Keyword Debug en commentaires lors de la création de l'exécutable final tout en ayant la possibilité 
  de tracer facilement l'exécution du programme pour le développeur. @LineBreak
  @LineBreak
  'NiveauDebug' est le niveau de priorité des messages du débogueur. Tous les messages (avec un niveau 
  non spécifié) sont affichés automatiquement. Lorsqu'un niveau est spécifié alors le message correspondant 
  ne sera affiché que si le niveau de debug courant est égal ou supérieur au niveau associé au message. 
  Cela permet de réaliser une traçabilité hiérarchisée en affichant des informations de plus en plus 
  précises en fonction de la valeur 'NiveauDebug' utilisée. 
@LineBreak
@LineBreak

; -------------------------------------------------------------------------------------------------
@FormatIf HTML
<br><hr><br>
<a name="DebugLevel"></a>
@FormatEndIf

@Syntax

  @Keyword DebugLevel <expression constante >

@Description    

  Fixe le niveau courant pour les messages 'Debug'. 
  @LineBreak
  @LineBreak
  Note: Le niveau est fixé au moment de la compilation, ce qui signifie que vous devez mettre la commande
  @Keyword "DebugLevel" avant les commandes debug pour être sûr qu'elles seront bien toutes affectées.
@LineBreak
@LineBreak

; -------------------------------------------------------------------------------------------------
@FormatIf HTML
<br><hr><br>
<a name="DisableDebugger"></a>
@FormatEndIf

@Syntax

  @Keyword DisableDebugger

@Description

   Interromps l'utilisation du débogueur sur les lignes du code qui suivent cette commande. 
   Il ne désactive pas complètement le débogueur, donc les vérifications de performances ne doivent pas être 
   effectuées à l'aide de la commande DisableDebugger, mais en désactivant le débogueur avant de compiler le programme.
@LineBreak
@LineBreak

; -------------------------------------------------------------------------------------------------
@FormatIf HTML
<br><hr><br>
<a name="EnableDebugger"></a>
@FormatEndIf

@Syntax

  @Keyword EnableDebugger

@Description    

  Active l'utilisation du débogueur sur les lignes de code qui suivent cette commande (lorsque 
  le débogueur a été préalablement interrompu par la commande @Keyword DisableDebugger). 
@LineBreak
@LineBreak
